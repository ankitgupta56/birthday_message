💌 ADD YOUR OWN CONTENT (Super Easy!):
═══════════════════════════════════════════════════════════════════════════

Method 1: Browser Console (EASIEST)
───────────────────────────────────

1. Press F12 → Console tab
2. Paste this (modify message):

fetch('/api/lovenote', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    message: "You make my heart so happy",
    emoji: "💖"
  })
}).then(r => r.json()).then(d => console.log('Added!', d));

3. Press Enter → Done!
4. Your message now in database!

Method 2: Terminal/curl
──────────────────────

curl -X POST http://localhost:5000/api/lovenote \
  -H "Content-Type: application/json" \
  -d '{"message":"Your message","emoji":"❤️"}'
