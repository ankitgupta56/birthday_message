## 🎨 CUSTOMIZATION

### Change Password
Edit `.env`:
```env
WEBSITE_PASSWORD=your-new-password
```

### Replace AI Images with Your Photos
In `public/index.html`, find `photosDataEnhanced` array and update image URLs or add local image paths.

### Personalize Name
In `public/index.html`, change:
```javascript
const herName = "Priya";  // Change to your name
```

### Update Messages
Edit the various message arrays in `public/index.html`:
- `loveNotes`
- `loveMessages`
- `promises`
- `dreams`
- `whatYouMean`
- `reminders`
