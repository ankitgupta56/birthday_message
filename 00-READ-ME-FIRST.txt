🚀 GETTING STARTED (3 SIMPLE STEPS)
================================================================================

STEP 1: Set Up MongoDB
Choose ONE option below:

Option A: Local MongoDB (Easiest for testing)
  1. Download: https://www.mongodb.com/try/download/community
  2. Install on your computer
  3. Start the MongoDB service
  4. The .env is already configured!

Option B: MongoDB Atlas (Free cloud database)
  1. Create account: https://www.mongodb.com/cloud/atlas
  2. Create free cluster
  3. Copy connection string
  4. Edit .env and replace MONGO_URI with your string
  5. Save .env

STEP 2: Start Your Backend Server
Open Terminal and run:

  cd path/to/birthday.html
  npm start

You should see:
  ✅ MongoDB Connected!
  🎁 Birthday Love Server running on http://localhost:5000

STEP 3: Visit Your Website
Open browser and go to:

  http://localhost:5000

Done! The site loads with sample data!

💻 HOW TO ADD CONTENT
================================================================================

EASIEST: Browser Console
1. Open http://localhost:5000
2. Press F12
3. Click Console tab
4. Run:

fetch('/api/lovenote', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    message: "Your message",
    emoji: "❤️"
  })
}).then(r => r.json()).then(d => console.log('Added:', d));

5. Press Enter - done!

🆘 TROUBLESHOOTING
================================================================================

"MongoDB Connection Error"
→ Is MongoDB running?
→ Check MONGO_URI in .env
→ For Atlas, whitelist your IP

"Port 5000 already in use"
→ Change PORT in .env to 3000 or 8000

"CSS/JS not loading"
→ Clear browser cache (Ctrl+Shift+Delete)
→ Check Console (F12)
